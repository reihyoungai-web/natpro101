    🎮 ASNP NATPRO10 in JEJU Island

    제주도를 배경으로 한 8비트 레트로 스타일 미니 게임입니다.

    📋 프로젝트 개요

    • 게임명: ASNP NATPRO10 in JEJU Island
    • 장르: 캐주얼 아케이드 게임
    • 플랫폼: 웹 (PC, 모바일, 태블릿 모두 지원)
    • 특징: 8비트 레트로풍 그래픽, 제주도 테마

    🎯 게임 규칙

    1. 돌하르방을 터치/클릭하여 위로 던집니다
    2. 날아다니는 한라봉을 수집하면 +10점
    3. 문어나 거북이에 닿으면 게임 오버!
    4. 점점 빨라지는 난이도에서 최고 점수에 도전하세요

    🌐 페이지 구성

    페이지 경로 설명
    게임 메인 /index.html 게임 플레이 페이지
    관리자 /admin.html 데이터 조회 및 CSV 다운로드

    ✅ 구현된 기능

    1. 초기화면

    • 8비트 레트로풍 제주도+바다 배경
    • 게임 타이틀 중앙 표시
    • "touch to throw" 문구 1초 간격 깜빡임
    • 8비트 스타일 BGM 재생
    • 터치/클릭으로 다음 화면 이동

    2. 이메일 입력 화면

    • 이메일 입력 폼
    • 이메일 유효성 검증
    • 입력된 이메일 서버에 저장

    3. 게임 플레이

    • 돌하르방 캐릭터 (플레이어)
    • 한라봉 아이템 수집 (+10점)
    • 문어/거북이 장애물
    • 제주도 배경 (한라산, 바다)
    • 점진적 난이도 증가
    • 실시간 점수 표시

    4. 게임 종료 & 랭킹

    • 최종 점수 표시
    • TOP 10 랭킹 보드
    • 점수 데이터베이스 저장
    • 재시작 기능

    5. 관리자 기능

    • 전체 플레이 데이터 조회
    • 통계 대시보드
    • CSV 파일 다운로드 (Excel 호환)
    • JSON 파일 다운로드

    ❌ 구현되지 않은 기능

    • 자동 이메일 전송: 매일 오후 8시 자동 이메일 전송은 서버 측 기능이 필요하여 정적 웹사이트에서는 구현 불가
    • 대안: 관리자 페이지에서 직접 CSV 다운로드 후 수동 이메일 전송

    📁 파일 구조

    /
├── index.html          # 메인 게임 페이지
├── admin.html          # 관리자 페이지
├── css/
│   └── style.css       # 8비트 레트로 스타일
├── js/
│   ├── audio.js        # 8비트 사운드 시스템
│   ├── game.js         # 게임 로직 및 렌더링
│   └── main.js         # 앱 컨트롤러
└── README.md           # 프로젝트 문서

    🗄️ 데이터 모델

    players 테이블

    필드명 타입 설명
    id text 고유 식별자 (자동 생성)
    email text 플레이어 이메일
    score number 게임 점수
    played_at datetime 플레이 시간

    🚀 사용 방법

    게임 플레이

    1. 게임 페이지(index.html)에 접속
    2. 화면을 터치/클릭하여 시작
    3. 이메일 주소 입력
    4. 게임 시작!
    5. 화면을 터치/클릭하면 돌하르방이 위로 점프
    6. 한라봉을 수집하고 문어/거북이를 피하세요

    데이터 다운로드

    1. 관리자 페이지(admin.html)에 접속
    2. "CSV 다운로드" 버튼 클릭
    3. 다운로드된 파일을 Excel에서 열기

    🔧 기술 스택

    • HTML5: 시맨틱 마크업
    • CSS3: 8비트 레트로 디자인, 반응형 레이아웃
    • JavaScript (ES6+): Canvas API, Web Audio API
    • 데이터 저장: RESTful Table API

    📱 지원 기기

    • ✅ iPhone (Safari)
    • ✅ Android (Chrome)
    • ✅ PC (Chrome, Firefox, Safari, Edge)
    • ✅ 태블릿

    🔒 보안

    • 클라이언트 측 이메일 유효성 검증
    • 랭킹 표시 시 이메일 마스킹 (예: abc***@gmail.com)
    • HTTPS를 통한 데이터 전송 (배포 시)

    📝 향후 개선 사항

    1. 더 다양한 아이템 추가
    2. 파워업 시스템
    3. 소셜 공유 기능
    4. 다국어 지원
    5. 오프라인 플레이 지원 (PWA)

    📖 사용 매뉴얼 (초보자용)

    게임 배포하기

    1. Publish 탭으로 이동

      • 화면 상단의 "Publish" 탭을 클릭하세요

    2. 배포하기

      • "Publish" 버튼을 클릭하면 자동으로 배포됩니다
      • 배포가 완료되면 URL이 제공됩니다

    3. URL 공유

      • 제공된 URL을 복사하여 공유하세요
      • 모바일에서도 해당 URL로 접속하면 게임을 플레이할 수 있습니다

    데이터 다운로드 방법

    1. 관리자 페이지 접속

      • 배포된 URL 뒤에 /admin.html을 추가하여 접속
      • 예: https://your-site.com/admin.html

    2. CSV 다운로드

      • "CSV 다운로드" 버튼 클릭
      • 파일이 자동으로 다운로드됩니다

    3. Excel에서 열기

      • 다운로드된 .csv 파일을 Excel에서 열기
      • 이메일과 점수 데이터가 표시됩니다

    게임 조작 방법

    기기 조작 방법
    PC 마우스 클릭
    모바일/태블릿 화면 터치

    문제 해결

    소리가 안 나요

    • 화면 오른쪽 상단의 🔊 아이콘을 확인하세요
    • 🔇로 표시되면 클릭하여 소리를 켜세요
    • 모바일의 경우 무음 모드를 해제하세요

    게임이 느려요

    • 다른 탭이나 앱을 종료해보세요
    • 페이지를 새로고침해보세요

    점수가 저장 안 돼요

    • 인터넷 연결을 확인하세요
    • 올바른 이메일 형식인지 확인하세요
